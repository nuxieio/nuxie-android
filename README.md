# Nuxie Android SDK

Kotlin/Android reference implementation of the Nuxie client SDK.

This repo is intended to be used as a git submodule from the Nuxie monorepo under `packages/nuxie-android`.

## Development

- JDK 17+
- Android SDK (compileSdk 34)

### Build

```sh
./gradlew build
```

### Tests

```sh
./gradlew test
```
